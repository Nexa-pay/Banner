# Telegram Report Bot

A Telegram bot that allows users to report channels, groups, and users. Easy to deploy on Railway.

## Features

- Report users, groups, and channels
- Multiple report reasons
- Cooldown system to prevent spam
- Admin notifications
- Report confirmation
- Easy deployment on Railway

## Deployment on Railway

1. **Create a Telegram Bot**
   - Message [@BotFather](https://t.me/botfather) on Telegram
   - Create a new bot with `/newbot`
   - Copy the bot token

2. **Fork/Clone this Repository**
   ```bash
   git clone https://github.com/yourusername/telegram-report-bot.git
   cd telegram-report-bot
